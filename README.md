3D FPS Game made for BetaJam 2025 in Godot 4.4

# This Game includes:
- Walking,
- Jumping
- and Sprinting!

# Controls
- WASD to move
- Space to jump
- Left Shift to sprint
- ~~Left Control to crouch~~
- E to pick up items
