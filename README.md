3D FPS Game made for BetaJam 2025 in Godot 4.4

# This Game includes:
- Walking,
- Jumping
- Sprinting
- Grab items
- Hooking

# Controls
- WASD to move
- Space to jump
- Left Shift to sprint
- ~~Left Control to crouch~~
- E to grab items
- Hold Left Click to hook.
- Use number keys to select items from hotbar.
